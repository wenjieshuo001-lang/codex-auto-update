# 音乐创作助手 (Music Creator)

## 概述

AI 辅助音乐创作工具。核心原则：**AI 生成灵感 → 人工实质改编 → 成为真正的原创作品**。

不是"把AI歌曲伪装成原创"，而是"用AI加速创作流程，让每一首歌都真正属于你"。

## 工作流程

### 1. 灵感生成 (AI辅助)
- 生成和弦进行方案 (C大调、A小调等)
- 提供旋律走向建议
- 歌词主题和结构建议
- 编曲风格参考

### 2. 人工改编 (核心步骤)
将 AI 输出作为**起点**，进行实质修改：
- **改和弦** — 替换 30%+ 的和弦，改变进行
- **改旋律** — 重新编排旋律线，调整节奏型
- **改歌词** — 重写/大幅度修改歌词内容和结构
- **改编曲** — 更换乐器配置、节奏风格、段落结构

### 3. 录制与制作
- 使用 DAW (FL Studio/Cubase/Logic) 重新录制所有 MIDI/音频轨道
- 混音、母带处理
- 导出符合平台标准的音频文件

### 4. 发布流程
- 网易云音乐/QQ音乐 创作者入驻
- 上传经过实质改编的作品
- 按要求进行 AI 辅助创作标识

## 评估标准

一份改编作品可以称为"原创"的参考标准：
- [ ] 和弦进行与原 AI 输出差异 > 50%
- [ ] 旋律线经过人工重写
- [ ] 歌词经过实质修改
- [ ] 编曲经过重新设计
- [ ] 所有音频轨道为人工录制/MIDI编排
